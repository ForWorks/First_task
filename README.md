Создать проект в Android studio по шаблону Empty Activity.

Подключить VCS git к Вашему проекту в студии, затем запушить его в bitbucket с описанием коммита “First commit”.

Для текста “Hello World!”, присутствующего на форме, созданной шаблоном, добавить локализацию на выбор на 8-ми разных языках.

Для версий Android выше 7 сделать цвет этого текста синим, для более низких - красным (тоже с помощью ресурсов).

Скачать с https://material.io/resources/icons/ любую иконку в формате png для Android (скачается архив с набором иконок), добавить в проект.

Установить скачанную иконку в конце текста все в той же TextView.

Запушить изменения в git c описанием коммита “Home task 1”.
